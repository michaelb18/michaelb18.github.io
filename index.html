Baseballbot

Baseballbot a tool I use to project the statistics of baseball players. This project consists of several Tensorflow models, each of which are trained to project one statistic. For example, one model predicts how many home runs a player will hit per year, another predicts how many runs batted in he will get and so on for every statistic. I use the project to gain an advantage in my fantasy baseball league, and I've had a lot of success so far in initial testing of the projectors.

Infinitam

This is my Microsoft Imagine Cup 2019 team. We are one of the Archimedes design teams in the Galipatia Living and Leaning community at Virginia Tech. The team is working on an app on iOS that evaluates the health of bee hives. The main mission behind the project is to provide an easy way to evaulate hive health to hobbyist bee keepers. I am working on making a deep learning model to evaluate the bee health for the app.

VT Baja Testing Team

I help program the sensor suite for the Virginia Tech Baja 2019 Car. Specifically, I coded the data reading software in C++ for the Intertial Measurement Unit(IMU), Linear Potentiometer, and several other sensors. I also helped make software to write data from the sensors to a CSV file.

I'm also working on VT Baja's secure storage project, which is an inventory management system for drill bits. I used the Google Drive API to write software in python that could upload data from the inventory management system to a Google sheet.
